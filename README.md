## Object Detection Prototype using YOLOv8 Nano
This project is a learning prototype for object detection using a pretrained YOLOv8 Nano model in Python. The goal was to gain hands-on experience with object detection workflows, image preprocessing, and model inference, experimenting with YOLOv8.

## Libraries Used
Python 3.x
Ultralytics YOLOv8
OpenCV
NumPy

## Project Description
Implemented a basic object detection pipeline using a pretrained YOLOv8 Nano model.
Performed image preprocessing including resizing and normalization for improved model input.
Generated bounding boxes and class labels for objects in sample images.
Explored model inference, output interpretation, and visualization techniques.

## Learning Outcomes
Gained practical experience with YOLOv8 architecture and workflow.
Learned how to run pretrained models for object detection tasks.
Developed skills in image preprocessing and bounding box visualization.
Learned to handle different model outputs and understand experimental results.
